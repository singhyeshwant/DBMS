1. `Model.pdf`: The final ER model made out of the data tables after applying Normalisation. This diagram is created using 'Lucidchart: Intelligent Diagramming' (https://www.lucidchart.com/pages/).
2. `Database.sql`: SQL queries to create empty Tables based on Relational Database Schema generated from the ER Model. These tables are added to the Database 'Vaccinations.db'. Later, the data is also added to its respective table manually.
3. `Queries.sql`: SQL queries performing Data Retrieval and Visualisation.
4. `Queries.pdf`: PDF of SQL queries performing Data Retrieval and Visualisation.
5. `Vaccinations.db`: The finally created Database.
